  • posicionamiento:lo que hace es posicionar elementos absoluta y relativamente en la página
  • position relative:El elemento sigue ocupando espacio como si fuera estático, pero la caja generada se puede desplazar a otra posición
  • position aboslute :El posicionamiento absolute hace que un elemento se coloque respecto a su contenedor posicionado mas cercano
  • position fixed:Le dices dónde posicionarse al elemento, y se queda ahí mientras el usuario hace scroll. El elemento no conserva su espacio en el flujo, todo el espacio que tuviera reservado ese elemento desaparece
  • top:La propiedad top especifica parte de la posición de un elemento posicionado. No tiene efecto en elementos no posicionados
  • right:representa todas las páginas de la derecha de un documento impreso. Que una página dada sea "izquierda" o "derecha" está determinada por la dirección principal de escritura del documento
  • bottom :es la propiedad de un elemento que establece el espacio requerido en la parte de abajo de un elemento.Tambien se permiten valores negativos
  • left :el left es como el right pero inverso es decir que La dirección principal de escritura del documento determina si una página es "izquierda" o "derecha"
  • z index:La propiedad CSS z-index indica el orden de un elemento posicionado y sus descendientes. Cuando varios elementos se superponen, los elementos con mayor valor z-index cubren aquellos con menor valor como el padre al hijo mayor
